# Come Iniziare — STEELEX Cantieri App

## Primo avvio (una sola volta)

### 1. Apri Claude Code desktop app
Apri l'app Claude Code sul tuo PC.

### 2. Aggiungi questa cartella
Cerca "Add folder" o "Open project" nell'app
e seleziona questa cartella: `C:\steelex-cantieri-cloud`

### 3. Scrivi questo primo messaggio
```
Sei pronto. Leggi il CLAUDE.md e inizia dalla Fase 1:
setup completo del progetto con deploy su Railway.app.
Vai in autonomia e dimmi quando il backend è online.
```

### 4. Aspetta e osserva
Claude Code farà tutto da solo:
- Creerà tutta la struttura del codice
- Configurerà il deploy su Railway
- Ti dirà solo quando ha bisogno delle tue API key

## Per le sessioni successive
Apri Claude Code nella cartella e digli cosa vuoi:
- "Aggiungi la funzione upload documenti"
- "Crea il modulo fatture fornitori"  
- "Fai funzionare le notifiche push"

Lui sa già tutto il resto dal CLAUDE.md.

## Costi cloud (Railway.app)
- Piano gratuito: ottimo per sviluppo e test
- Piano Pro (~$5/mese): per uso reale produzione
- Database PostgreSQL: incluso nel piano
